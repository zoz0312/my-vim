# my-vim
> Vim 설정값 백업

## Setup
```
#복사하는 경우
$ git clone https://github.com/zoz0312/my-vim.git ~/my-vim
$ cp ~/my-vim/vimrc ~/.vimrc
$ cp -R ~/my-vim/vim ~/.vim

#다운받는경우
$ git clone https://github.com/zoz0312/my-vim.git ~/my-vim
$ cp ~/my-vim/vimrc ~/.vimrc
$ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
$ vi ~/.vimrc
#PluginInstall

$ vi ~/.vim/bundle/vim-closetag/plugin/closetag.vim
#12: call s:Declare('g:closetag_filetypes', 'html,xhtml,phtml,vue')
#15: call s:Declare('g:closetag_filenames', '*.html,*.xhtml,*.phtml,*.vue')

```
